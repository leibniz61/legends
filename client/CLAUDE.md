## Styling

- Default to using shadcn components whenever they are applicable
- Use tailwind units (e.g. gap-2, p-4, etc.)
- The default unit in tailwind is set to 0.25rem or 4px (e.g. gap-2 == 0.5rem or 8px, p-4 = 1rem or 16px)
- Spacing, margins, padding, corners should usually be divisible by 4 (e.g. gap-2, p-4, etc.)
- Use px for border width (usually 0.5px or 1px, e.g. border-[0.5px])
